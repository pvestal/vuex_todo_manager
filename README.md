# This Codebase

> This project is to manage code snippets

## use nvm version recommended by Vue

```
nvm install 8.11.0
```

### install dependencies

```
npm install
```

### vue cli

```
npm install -g @vue/cli
```

### firebase cli

```
npm install -g firebase-tools
```

### firebase init

```
firebase init
```

### vue build

```
npm run build
```

### firebase deploy

```
firebase deploy
```
